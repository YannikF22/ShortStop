Short Stop is a URL blocking Chrome extension to stop you from doom scrolling YouTube Shorts.
It has three modes, available from the extension drop down menu:
	Soft lock (standard): You can watch any short, but you are blocked from scrolling to the next one. It may take a moment to reload the URL of the first one, so the second one could play for a moment.
	Hard lock: All YT short URLs are blocked.
	Unlock: All unlocked, reverts to soft lock after 5min
	To fully deactivate use the extensions menu to toggle it off.
	
To install just put the files in a folder on your computer, then click "add unpacked extension" in the chrome extensions menu and choose the folder.
